Icon systems · 6 librerías comparadas

Nivel de calidad por librería

Mismos 6 iconos (workout/check-in/medalla/fotos/level/coach) renderizados con diferentes sistemas. Escoge según calidad visual, licencia y esfuerzo de integración.

Librería Workout Check-in Medalla Fotos Level Coach Notas
HeroiconsMIT

Tailwind Labs · 316 iconos

 Ya en uso. Consistente. Stroke 1.5. Variante solid disponible. Baseline.
PhosphorMIT

6 pesos · 9,072 iconos

 Top-tier calidad. 6 pesos (thin/light/regular/bold/fill/duotone). Más visual que Heroicons.
LucideISC

Fork de Feather · 1,400+ iconos

 Muy liviano. 1 peso solo pero muy pulido. Similar filosofía a Heroicons.
SF SymbolsApple-only

6,900+ iconos Apple
⚠ Licencia propietaria de Apple. Solo permitido en apps que corren en plataformas Apple (iOS/macOS/visionOS). NO se pueden usar en web. Alternativa: SF Pro font (misma estética) es licenciable para web.
❌ No aplicable para web.
TablerMIT

4,876 iconos · 2 pesos

 Versión outline y filled. Íconos fitness específicos: barbell, dumbbell, stretching, run, yoga.
WellCore custombrand

~20 SVGs diseñados · 0 libs

 Máximo nivel de marca. Gradientes, profundidad, detalles específicos. Esfuerzo: diseñar ~20 iconos clave. Combina con Phosphor/Lucide para resto.

Recomendación técnica

Phosphor (MIT) + ~10 SVG custom WellCore

Phosphor cubre 95% de necesidades (UI general, acciones), y diseñamos ~10 iconos brand-specific para: hexágono medalla, trofeo XP, checkmark premium, confeti piece, racha llama, progreso ring. Todos en SVG optimizado con gradientes WC.

Opción "budget infinito"

Lottie animado para cada milestone

Contratar motion designer para ~15 animaciones Lottie JSON (level-up ring, medal unlock, confetti burst, etc). Se reproducen con lottie-web. Cada modal tiene su animación única de 2-3s. Esfuerzo: USD 200-400 por animación.

Matrix de decisión · cómo escoger

Qué combinación aplicamos

Perfil WellCore Estilo base Icon library Esfuerzo Impacto visual Mi pick
Mínimo viable Bento WC (actual) Heroicons (ya en uso) 1 día 8/10
Premium balanceado Bento WC + gradient heroes Phosphor + 10 SVG custom 3-4 días 9.5/10 ← recomendado
Aspiracional Liquid Glass + Bento hybrid Phosphor duotone + Lottie 1-2 semanas 10/10
Diferenciado raro Aurora mesh o Brutalist Custom art direction 2-3 semanas 9/10 + riesgo
Mi lectura honesta: Los estilos 3/5/7/8 (Clay, Terminal, Duo, Brutalist) no encajan con la marca WellCore adulta/premium — sirven como inspiración pero romperían coherencia. El estilo 2 (Liquid Glass iOS) es el más aspiracional pero requiere ajuste de paleta para mantener el rojo #DC2626 como protagonista. El estilo 6 (Aurora) es el más "2026-style" pero pierde la legibilidad de datos que Bento da. Mi recomendación es quedarnos en Bento WC pero subir su nivel con: iconos Phosphor duotone + 10 SVG brand-custom + animaciones sutiles (el ring ya hace eso). Eso da 95% del impacto de "Liquid Glass" con 30% del esfuerzo.

Type system · SF Pro aplicado

Todo el gallery ahora renderiza con SF Pro Display (títulos + data) y SF Pro Text (body). En Mac/iOS se resuelve nativo via -apple-system, en Windows/Android se cargan los WOFF desde sf.abarba.me. Siguiente paso: aplicar la misma familia en la plataforma real via resources/css/app.css.
