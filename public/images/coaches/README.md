# Coach Photos

Professional photos of WellCore coaches.

## Specifications
- **Format**: WebP preferred, PNG fallback
- **Dimensions**:
  - Profile: 400x400px (square)
  - Full body: 600x900px (portrait, 2:3 ratio)
  - Card: 300x300px (square, for dashboard cards)
- **Max file size**: 150KB
- **Naming**: coach-firstname-type.webp (e.g., silvia-profile.webp, silvia-full.webp)
- **Subdirectories**: Create a subdirectory per coach for multiple photos
