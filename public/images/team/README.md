# Team Photos

Photos of the WellCore Fitness team for the "Nosotros" page.

## Specifications
- **Format**: WebP preferred, PNG fallback
- **Dimensions**: 400x400px (square, will be displayed as circle)
- **Max file size**: 100KB
- **Naming**: firstname-lastname.webp (e.g., daniel-esparza.webp)
- **Background**: Transparent or dark neutral
- **Style**: Professional headshot, consistent lighting
