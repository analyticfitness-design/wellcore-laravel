# Client Transformations

Before/after photos of client transformations for social proof.

## Specifications
- **Format**: WebP preferred, JPG fallback
- **Dimensions**: 600x800px (portrait, 3:4 ratio)
- **Max file size**: 200KB per image
- **Naming**: client-name-before.webp / client-name-after.webp
- **Privacy**: Must have written client consent
- **Style**: Same pose, lighting, and background for before/after consistency
