## Instrucciones para ejecutar cada wave

```
Prompt por agente:
"Lee el blade original [BLADE_PATH], lee el PHP Livewire [PHP_PATH],
lee el Vue actual [VUE_PATH], y edita el Vue para agregar CADA feature
que el blade tiene y el Vue no. NO borres funcionalidad existente.
Usa useApi(), [Layout], WellCore design tokens."
```

Despues de cada wave:
1. `npx vite build` — verificar 0 errores
2. `git add [files] && git commit`
3. `git push origin main`
